AbleSpace is a real-time task management web application that allows users to create, assign, track, and manage tasks efficiently with instant notifications.

Features
User authentication (JWT)
Create & assign tasks
Role-based task status updates
Real-time dashboard updates (Socket.IO)
Instant notifications with unread count
Task filtering & sorting
Overdue task highlighting

Tech Stack
Frontend: React, TypeScript, Tailwind CSS
Backend: Node.js, Express, TypeScript
Database: MongoDB (Mongoose)
Real-time: Socket.IO

Deployment: Vercel

Setup
git clone https://github.com/Manikandan0018/AbleSpace.git
cd AbleSpace


Backend

cd backend
npm install
npm run dev


Frontend

cd frontend
npm install
npm run dev
